# Ambiguity Clustering An Accurate and Efficient Decoder for qLDPC Codes

## Summary
Introduces Ambiguity Clustering (AC) to decode qLDPC codes. It divides measurement data into independent clusters after Belief Propagation stalls, outperforming BP-OSD by 27x in speed.

## Method Details
- **Authors**: Stasiu Wolanski, Ben Barber
- **Published**: 2024-06-20
- **arXiv ID**: [2406.14527v2](https://arxiv.org/abs/2406.14527v2)

## Reproducibility Notes
Implemented on Bivariate Bicycle codes. Cluster identification is key.
