# UniDetect: LLM-Driven Universal Fraud Detection Across Heterogeneous Blockchains

**Authors:** Yufei Wang et al.
**Affiliations:** National University of Singapore (NUS), Nanyang Technological University
**Source:** [arXiv](https://arxiv.org/abs/2604.17555)
**Date:** 2026-04-16

## Summary
Cross-chain fraud is increasingly complex. UniDetect leverages LLMs to parse and understand heterogeneous smart contract bytecode and transaction traces across EVM and non-EVM chains, creating a universal semantic representation for anomaly detection.

## Implementation Notes
Added a prompt template and processing script for converting raw EVM traces into the universal semantic format described in the paper.
