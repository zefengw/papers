# Compressing Sequences in the Latent Embedding Space: K-Token Merging for Large Language Models
    
## Summary
Proposes K-Token Merging to compress sequences in the latent embedding space, reducing the number of tokens processed in subsequent layers without significant loss in performance.

## Method Details
Based on the paper https://arxiv.org/abs/2604.15153, the method proposes a way to proposes k-token merging to compress sequences in the latent embedding space, reducing the number of tokens processed in subsequent layers without significant loss in performance.

## Reproducibility Notes
- Implementation is provided in the `impl/` folder.
- Use Python 3.9+ and relevant libraries (PyTorch, NumPy).
