# How Do AI Agents Spend Your Money? Analyzing and Predicting Token Consumption in Agentic Coding Tasks

**Authors:** Longju Bai, Zhemin Huang, Xingyao Wang, Jiao Sun, Rada Mihalcea
**Affiliation:** University of Michigan
**Link:** [http://arxiv.org/abs/2604.22750v1](http://arxiv.org/abs/2604.22750v1)

## Summary
Empirical study and predictive modeling of token consumption in agentic workflows (e.g., software engineering agents). It introduces metrics to measure 'monetary efficiency' and tools for predicting cost before task execution.

## Method Details
A Python cost-prediction module for AI agents that estimates token usage based on the task difficulty, codebase size, and historically typical iterations for similar tasks.

## Reproducibility Notes
This implementation focus on the core algorithms described in the paper.
