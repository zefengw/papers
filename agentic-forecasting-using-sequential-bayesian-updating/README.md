# Agentic Forecasting Using Sequential Bayesian Updating of Linguistic Beliefs

**Authors:** Alexandre de Baynast et al.
**Affiliations:** MIT, Stanford University
**Source:** [arXiv](https://arxiv.org/abs/2604.19890)
**Date:** 2026-04-21

## Summary
LLM agents for forecasting represent a major emerging enterprise use case. This paper formalizes how agents can maintain and update probabilistic beliefs sequentially using language, bridging classical Bayesian updating with modern LLM capabilities.

## Implementation Notes
Created a lightweight Bayesian updating harness for LLMs that adjusts a probability distribution based on sequential text evidence.
