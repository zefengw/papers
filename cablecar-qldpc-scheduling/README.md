# CAbLECAR: Scheduling QLDPC Codes
Based on arXiv:2604.24739.

## Abstract Summary
CAbLECAR is a framework for scheduling operations in Quantum Low-Density Parity-Check (QLDPC) codes, optimizing for hardware constraints and minimizing execution time.

## Implementation
The `impl/` directory contains a toy scheduler demonstrating the CAbLECAR prioritization logic.
