# LaST-R1: Reinforcing Action via Adaptive Physical Latent Reasoning for VLA Models

## Summary
Integrates latent Chain-of-Thought (CoT) over physical dynamics into Vision-Language-Action models, using a novel RL algorithm (LAPO) to optimize reasoning and control simultaneously.
