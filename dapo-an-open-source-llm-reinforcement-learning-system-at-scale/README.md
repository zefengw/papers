# DAPO An Open-Source LLM Reinforcement Learning System at Scale

## Summary
Introduces Decoupled Clip and Dynamic Sampling Policy Optimization (DAPO) to stabilize large-scale RL for reasoning models. Open-sources a system achieving 50 on AIME 2024.

## Method Details
- **Authors**: Qiying Yu, et al.
- **Published**: 2025-03-18
- **arXiv ID**: [2503.14476v2](https://arxiv.org/abs/2503.14476v2)

## Reproducibility Notes
Built on the Verl framework. Key implementation details are in the asymmetry of the clip function and dynamic sampling of reward-heavy trajectories.
