# Absolute Zero Reinforced Self-play Reasoning with Zero Data

## Summary
Proposed the Absolute Zero paradigm where a model (AZR) self-evolves by proposing its own tasks and solving them using a code executor as a source of verifiable rewards, eliminating the need for human-annotated data.

## Method Details
- **Authors**: Andrew Zhao, Yiran Wu, Yang Yue, Tong Wu, Quentin Xu, Yang Yue, Matthieu Lin, Shenzhi Wang, Qingyun Wu, Zilong Zheng, Gao Huang
- **Published**: 2025-05-06
- **arXiv ID**: [2505.03335v3](https://arxiv.org/abs/2505.03335v3)

## Reproducibility Notes
Requires a sandboxed code execution environment to act as the ground truth reward signal. Model uses two roles: Proposer and Solver.
