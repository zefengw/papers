# Context Unrolling in Omni Models

## Summary
Presents "Omni", a unified multimodal model trained on text, images, videos, 3D geometry, and hidden representations. It identifies the "Context Unrolling" phenomenon, where the model explicitly reasons across multiple modal representations before producing output, leading to superior multimodal reasoning capabilities.

## Key Methods
- **Native Multimodal Training**: Shared latent space for diverse sensory inputs.
- **Context Unrolling**: Explicitly expanding modal representations in context during reasoning.
- **Unified Embedding**: Tokenizing 3D geometry and video alongside text.
