# Securing Elliptic Curve Cryptocurrencies against Quantum Vulnerabilities: Resource Estimates and Mitigations
    
## Summary
Provides detailed resource estimates for breaking 256-bit ECC using quantum computers and proposes mitigation strategies to secure blockchains against these vulnerabilities.
