# k-Maximum Inner Product Attention for Graph Transformers and the Expressive Power of GraphGPS
    
## Summary
Introduces k-MIP attention to improve the efficiency and expressivity of Graph Transformers, allowing for better handling of large-scale graphs by approximating the maximum inner product.

## Method Details
Based on the paper https://arxiv.org/abs/2604.03815, the method proposes a way to introduces k-mip attention to improve the efficiency and expressivity of graph transformers, allowing for better handling of large-scale graphs by approximating the maximum inner product.

## Reproducibility Notes
- Implementation is provided in the `impl/` folder.
- Use Python 3.9+ and relevant libraries (PyTorch, NumPy).
