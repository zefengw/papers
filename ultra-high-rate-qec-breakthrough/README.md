# Ultra-High Rate QEC: A Breakthrough in Fault-Tolerant Quantum Computing

**Authors:** Michael J. Gullans et al.
**Affiliations:** IBM Quantum, University of Maryland
**Source:** [arXiv](https://arxiv.org/abs/2604.18021)
**Date:** 2026-04-18

## Summary
Quantum Error Correction (QEC) overhead is the primary bottleneck for useful quantum computers. This paper proposes a new family of high-rate codes that significantly reduces the physical-to-logical qubit ratio while maintaining strong error suppression.

## Implementation Notes
Implemented the novel parity-check matrix generation algorithm for the proposed high-rate code family using numpy.
