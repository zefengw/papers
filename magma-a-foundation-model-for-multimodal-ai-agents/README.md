# Magma A Foundation Model for Multimodal AI Agents

## Summary
Magma is a foundation model for digital and physical agents, trained on SoM (Set-of-Mark) for image grounding and ToM (Trace-of-Mark) for video planning.

## Method Details
- **Authors**: Jianwei Yang, et al.
- **Published**: 2025-02-18
- **arXiv ID**: [2502.13130v1](https://arxiv.org/abs/2502.13130v1)

## Reproducibility Notes
Focuses on spatial-temporal intelligence for robotics and UI navigation. Uses SoM labels as intermediate coordinates.
