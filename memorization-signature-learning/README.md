# Learning the Signature of Memorization in Autoregressive Language Models
    
## Summary
Develops a transferable learned attack to detect if a specific sequence was memorized during the training of an autoregressive language model, moving beyond hand-crafted heuristics.

## Method Details
Based on the paper https://arxiv.org/abs/2604.03199, the method proposes a way to develops a transferable learned attack to detect if a specific sequence was memorized during the training of an autoregressive language model, moving beyond hand-crafted heuristics.

## Reproducibility Notes
- Implementation is provided in the `impl/` folder.
- Use Python 3.9+ and relevant libraries (PyTorch, NumPy).
