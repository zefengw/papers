# LLaDA 2.0: Unifying Multimodal Understanding and Generation with Diffusion Large Language Model

**Authors:** Tingyu Qu et al.
**Affiliations:** Tsinghua University, ShengShu Technology
**Source:** [arXiv](https://arxiv.org/abs/2604.20796)
**Date:** 2026-04-22

## Summary
Diffusion models for LLMs are gaining massive traction as an alternative to autoregressive generation. LLaDA 2.0 unifies multimodal understanding and generation into a single diffusion-based framework, marking a significant step towards native multimodal foundation models without modality-specific encoders/decoders.

## Implementation Notes
Added a simplified 1D continuous diffusion forward pass for token embeddings to demonstrate the core training objective.
